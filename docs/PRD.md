# Project Requirements Document (PRD)

## Confirmed Decisions
- **Placement**: Plane-only placement
- **Billboard**: Yes
- **Instance Behavior**: Single instance per `characterId` with Dup1 behavior
- **UI Layout**: UI1 toolbar
- **Pose Bar**: Text-only
- **Skins Switching**: Switchable after placement with fallback to `defaultPose`
- **Shadow**: Default ON
- **Undo/Redo**: Linear timeline Y including pose/skin/shadow/transform/place/delete/z-order but excluding lock and global UI settings

## Scope
- **Versions**: v0.1/v0.2
