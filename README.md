# NekoAR Project Overview
NekoAR is an ARCore-based 2D character photo app designed to bring characters to life in augmented reality, allowing users to create and share engaging photos with customizable features.

## Roadmap

### v0.1 Features
- **Minimum SDK Requirements**: minSdk 29  
- **Programming Language**: Kotlin  
- **License**: MIT  
- **Features**:
  - Pose only  
  - PNG sequence support  
  - Skins  
  - Per-character shadow toggle  
  - Lock excluded from undo  
  - Undo/redo includes pose/skin/shadow/transform/place/delete  
  - Plane-only placement  
  - Billboard
  - High-res photo export without UI  
  - Grid/level/timer  

### v0.2 Features
- **Action Management**:  
  - Action + Animated WebP import  
  - Dynamic photo Animated WebP export (3s at 24fps) without UI (includes countdown)
- **Recording Options**:  
  - Recording disables editing  

## Character Pack v1 Specifications
- **Format**: ZIP  
- **Contents**:  
  - Poses  
  - Skins  
  - Metadata in JSON schema
